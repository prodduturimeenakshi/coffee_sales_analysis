# ☕ Coffee Sales Analysis Project

## 📌 Objective

This project analyzes coffee sales data to understand:

* Popular products
* Revenue generation
* Sales patterns over time
* Customer payment behavior

---

## 🛠 Tools Used

* Python
* Pandas
* Matplotlib

---

## 📊 Key Insights

* The most sold coffee is **AMERICANO WITH MILK**.
* The highest revenue-generating coffee is **LATTEE**.
* Sales are concentrated during specific time intervals (40-minute groups).
* Most customers prefer **CARD** payment method.

---

## 📈 Visualizations

This project includes:

* Top selling coffee chart
* Revenue analysis chart
* Time-based sales distribution
* Payment method analysis

---

## 📌 Conclusion

This analysis highlights customer preferences, peak sales periods, and revenue-driving products. These insights can help improve inventory planning, marketing strategies, and customer experience.
